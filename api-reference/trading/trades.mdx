The Trades API provides authenticated access to your execution records. You can query recent trades (last 24 hours), look up a specific trade by ID, or retrieve up to 90 days of historical trade data with datetime filtering. All endpoints require a valid bearer JWT.

---

## GET /trading-api/v1/trades

Returns a list of trades for the authenticated trading account based on specified filters. Only the last 24 hours of data is available. Supports [pagination](/concepts/pagination) and [filtering](/concepts/filtering) on `createdAtDatetime` and `createdAtTimestamp`.

<Note>
  **Authentication:** Required. **Rate limited:** Yes.
</Note>

### Query Parameters

<ParamField query="tradingAccountId" type="string" required>
  ID of the trading account.
</ParamField>

<ParamField query="symbol" type="string" required={false}>
  Filter by market symbol, e.g. `BTCUSD`.
</ParamField>

<ParamField query="orderId" type="string" required={false}>
  Filter by unique order ID.
</ParamField>

<ParamField query="otcTradeId" type="string" required={false}>
  Filter by unique Bullish OTC trade ID.
</ParamField>

<ParamField query="clientOtcTradeId" type="string" required={false}>
  Filter by unique client OTC trade ID.
</ParamField>

<ParamField query="_pageSize" type="integer" required={false}>
  Page size. One of: `5`, `25`, `50`, `100`. Defaults to `25`.
</ParamField>

### Response Fields (array of Trade objects)

<ResponseField name="tradeId" type="string">
  Unique exchange-assigned trade ID.
</ResponseField>

<ResponseField name="orderId" type="string">
  The order ID that generated this trade.
</ResponseField>

<ResponseField name="symbol" type="string">
  Market symbol. E.g. `BTCUSD`.
</ResponseField>

<ResponseField name="price" type="string">
  Execution price. E.g. `"50000.0000"`.
</ResponseField>

<ResponseField name="quantity" type="string">
  Executed quantity. E.g. `"0.50000000"`.
</ResponseField>

<ResponseField name="quoteAmount" type="string">
  Quote quantity deducted from the asset account (price × quantity net of fees).
</ResponseField>

<ResponseField name="baseFee" type="string">
  Base asset fee charged for this trade. E.g. `"0.00050000"`.
</ResponseField>

<ResponseField name="quoteFee" type="string">
  Quote asset fee charged for this trade. E.g. `"0.0005"`.
</ResponseField>

<ResponseField name="side" type="string">
  Trade side: `BUY` or `SELL`.
</ResponseField>

<ResponseField name="isTaker" type="boolean">
  Whether the account was the taker on this trade.
</ResponseField>

<ResponseField name="tradeRebateAmount" type="string">
  Amount of rebate credited to the account as part of this trade.
</ResponseField>

<ResponseField name="tradeRebateAssetSymbol" type="string">
  Asset symbol in which the rebate is paid.
</ResponseField>

<ResponseField name="otcMatchId" type="string">
  Unique OTC match ID (if applicable).
</ResponseField>

<ResponseField name="otcTradeId" type="string">
  Unique Bullish OTC trade ID (if applicable).
</ResponseField>

<ResponseField name="clientOtcTradeId" type="string">
  Unique client OTC trade ID (if applicable).
</ResponseField>

<ResponseField name="createdAtDatetime" type="string">
  ISO 8601 datetime when the trade was executed by the exchange.
</ResponseField>

<ResponseField name="createdAtTimestamp" type="string">
  Millisecond EPOCH timestamp when the trade was executed.
</ResponseField>

### Example Request

```bash
curl -X GET "https://api.exchange.bullish.com/trading-api/v1/trades?tradingAccountId=111000000000001&symbol=BTCUSD" \
  -H "Authorization: Bearer <JWT>"
```

<Note>

</Note>

### Path Parameters

<ParamField path="tradeId" type="number" required>

</ParamField>

### Query Parameters

<ParamField query="tradingAccountId" type="string" required>

## GET /trading-api/v1/history/trades

Returns a paginated list of historical trades for the authenticated trading account. Only the last 90 days of data are available. Supports [pagination](/concepts/pagination) and [filtering](/concepts/filtering) on `createdAtDatetime` and `createdAtTimestamp`.

<Note>
  **Authentication:** Required. **Rate limited:** Yes.
</Note>

### Query Parameters

<ParamField query="tradingAccountId" type="string" required>
  ID of the trading account.
</ParamField>

<ParamField query="symbol" type="string" required={false}>
  Filter by market symbol. E.g. `BTCUSD`.
</ParamField>

<ParamField query="orderId" type="string" required={false}>
  Filter by order ID.
</ParamField>

<ParamField query="tradeId" type="string" required={false}>
  Filter by specific trade ID.
</ParamField>

<ParamField query="otcTradeId" type="string" required={false}>
  Filter by Bullish OTC trade ID.
</ParamField>

<ParamField query="clientOtcTradeId" type="string" required={false}>
  Filter by client OTC trade ID.
</ParamField>

<ParamField query="createdAtDatetime[gte]" type="string" required={false}>
  Start of the datetime range. ISO 8601 with millisecond, e.g. `2024-09-01T00:00:00.000Z`.
</ParamField>

<ParamField query="createdAtDatetime[lte]" type="string" required={false}>
  End of the datetime range. ISO 8601 with millisecond.
</ParamField>

<ParamField query="_pageSize" type="integer" required={false}>
  Page size. One of: `5`, `25`, `50`, `100`. Defaults to `25`.
</ParamField>

<ParamField query="_nextPage" type="string" required={false}>
  Cursor to retrieve the next page, returned in paginated responses.
</ParamField>

<ParamField query="_previousPage" type="string" required={false}>
  Cursor to retrieve the previous page, returned in paginated responses.
</ParamField>

### Example Request

```bash
curl -X GET "https://api.exchange.bullish.com/trading-api/v1/history/trades?tradingAccountId=111000000000001&symbol=BTCUSD&createdAtDatetime%5Bgte%5D=2024-09-01T00%3A00%3A00.000Z&createdAtDatetime%5Blte%5D=2024-09-30T23%3A59%3A59.000Z" \
  -H "Authorization: Bearer <JWT>"
```
